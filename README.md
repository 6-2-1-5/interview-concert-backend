# Concert - Backend

## Prerequisites

- Node.js 18+
- pnpm package manager

## Getting Started

### 1. Install Dependencies

```bash
pnpm install
```

### 2. Compile and run the project

```bash
# development
$ pnpm run start
```

### 3. Service running [http://localhost:8080](http://localhost:8080).

## Run tests

```bash
# unit tests
$ pnpm run test

# test coverage
$ pnpm run test:cov
```
